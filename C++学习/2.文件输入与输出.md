# 控制台I/O
## cin
- ### 用字符(char)作为接收参数：
```c++
char ch;
cout << "输入: ";
cin >> ch;
cout << "ch = " << ch;
```
输入与输出：
```shell
$ 输入: 12.5 19.8
$ ch = 1 
```
- ### 用整型(int)作为接收参数:
```c++
int n;
cout << "输入: ";
cin >> n;
cout << "n = " << n;
```
输入与输出:
```shell
$ 输入: 12.5 19.8
$ n = 12
```
- ### 用双精度浮点(double)作为接收参数:
```c++

```
输入与输出:
```shell

```
- ### 用char数组作为接收参数:
1. #### cin直接传值
```c++
char ch[20];
cout << "输入:";
cin >> ch;
cout << "ch = " << ch;
```
输入与输出:
```shell
$ 输入: 12.5 19.8
$ ch = 12.5
```
2. #### getline传值
```c++
char ch[20];
cout << "输入:";
cin.getline(cin,20);
cout << "ch = " << ch;
```
输入与输出:
```shell
$ 输入: 12.5 19.8
$ ch = 12.5 19.8
```
